## Install

Clone the repo:

```bash
git clone https://github.com/satyam-sp/test-node-app.git
cd nodejs-postgresql-boilerplate
```

Install the dependencies:

```bash
npm install
```

## Commands

Start API server:

```bash
npm start
````

```bash
npm run dev
```
